# 🍽️ Restaurant Website

A **responsive restaurant website** showcasing menu items, gallery, special dishes, and contact information. Built with **HTML, CSS, and JavaScript**, this project focuses on clean UI, smooth navigation, and a modern layout.


## 🚀 Features

- **Home Section:** Hero banner with call-to-action button and tagline.  
- **About Section:** Story of the restaurant and mission statement.  
- **Special Dishes:** Images and descriptions of signature dishes.  
- **Gallery:** Photo gallery of meals and restaurant ambience.  
- **Contact Section:** Phone, email, address, and social media links.  
- **Footer:** Navigation links, social media icons, and logo.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile.  


## 🛠️ Technologies Used

- **HTML5** – structure of the website  
- **CSS3** – styling, layout, and responsiveness  
- **JavaScript** – interactive elements (if any)
